# Book Tracker API
This Rails app is the server for [Book Tracker client](https://github.com/fpezcara/book-tracker)


## Local development

#### To run tests:
```
bin/rails test
```

or
```
bin/rake
```

#### To run migration:
```
bin/rails db:environment:set RAILS_ENV=development
```

#### To check migration status:
```
bin/rails db:migrate:status
```

#### To start the server:
```
bin/rails server
```
